# Práctica Front-End – Gael Borchardt

Generate a Cat API

## Descripción

Pagina web para generar una foto random de un gato, gif random de un gato con o sin texto.
Haciendo requests a la API Cat as a Service (https://cataas.com) utilizando circuit breaker para evitar errores.



## Tecnologías utilizadas

* HTML5
* Tailwind
* Api Call
* JS

## Contenido del repositorio

* Estructura HTML semántica y accesible
* Hojas de estilo CSS personalizadas
* Script con funcionalidad
* Documentación básica del proyecto
