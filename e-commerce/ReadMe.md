# Prácticas Front-End – Gael Borchardt

Menu de Burgirs

## Descripción

Proyecto de práctica de página web estática.
Muestra un menú de productos de una tienda de hamburguesas, cargando datos desde un json de manera dinámica y
contando con la funcionalidad de añadir los productos al carrito.

## Tecnologías utilizadas

* HTML5
* CSS3
* JS

## Contenido del repositorio

* Estructura HTML semántica y accesible
* Hojas de estilo CSS personalizadas
* Script con funcionalidad
* Documentación básica del proyecto
